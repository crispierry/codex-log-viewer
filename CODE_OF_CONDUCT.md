# Code Of Conduct

We want Codex Log Viewer to be a useful and welcoming open source project.

Please be respectful, constructive, and careful with other people's data. This project deals with logs that may contain sensitive local development history, so privacy-conscious behavior is part of being a good contributor here.

Unacceptable behavior includes harassment, personal attacks, publishing private information, sharing unsanitized session logs, or pressuring others to reveal proprietary data.

If you need to report a problem, use the contact path listed in [SECURITY.md](SECURITY.md) for sensitive concerns or open a GitHub issue for ordinary project conduct concerns.
